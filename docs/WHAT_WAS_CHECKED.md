# Что проверено

Цель — визуально показывать пользователю начало с 1, но не менять формат Betaflight.

Проверочная логика:

- `0 1 2 3` в Betaflight показывается как `1 2 3 4`.
- При сохранении `1 2 3 4` возвращается в `0 1 2 3`.
- `25 200 500 800` остаётся `25 200 500 800` и не сдвигается.

Правка применяется к `src/js/tabs/vtx.js`.
Общее правило: сдвиг включается только когда таблица выглядит как zero-based control values: есть `0`, минимум `0`, максимум не больше количества уровней.
