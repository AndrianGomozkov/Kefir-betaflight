# Что проверено

1. Целевая версия: Betaflight App `2025.12.2`, commit `a2d0f50`.
2. Для этой версии существует старый файл вкладки VTX: `src/js/tabs/vtx.js`.
3. `package.json` версии `2025.12.2` требует Node `20.x`.
4. Сборка выполняется командой `npm run build`.
5. В `vite.config.js` не найден кастомный `outDir`, поэтому workflow сначала ищет `app/dist`, а потом запасной `app/src/dist`.
6. Правка сделана только на границе UI:
   - чтение в поле: `0 -> 1`, `1 -> 2`, `2 -> 3`, `3 -> 4`;
   - сохранение из поля: `1 -> 0`, `2 -> 1`, `3 -> 2`, `4 -> 3`.
7. Сдвиг включается только для последовательных маленьких таблиц `[0, 1, 2, 3...]`.
8. Таблицы типа `[25, 200, 500, 800]` не меняются.
